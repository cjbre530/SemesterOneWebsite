Commands


Varibles - (DIM)

"DIM" Varibles allow you to create varible names, and utilize it for its indivisual function. These varibles can be used for either integers or strings, creating an easier advantage for clean-up.

EXAMPLE

Syntax:

Dim [variable name] As [data type: Integer, Double, Boolean, String, Decimal, …]

Dim choice as String

Dim month as String

Dim stone as String


Decisions (If/Then and Select Case)

The If, Then Statement lets you make a decision between a String and or Integer. It can be used for any statement that you choose to have an immediate answer to after. "Select Case" is also used for the same result, but in a different way, as shown below.

EXAMPLE

(If, Then Statement)

Syntax:

if [condition] then

[else]

end if

EXAMPLE

(Select Case Statement)

Console.WriteLine("What is Your Birth Month?")

choice = Console.ReadLine()

select choice

case "January":

month = "January"

stone = "Garnet"


Loops - (for)

"For Loops" are created for a repatition of a String or Integer that you want repeated until it hits a certain value.

EXAMPLE

Syntax:

for [variable name and type = start to finish]

Statements

[exit for]

Next


Input (ReadLine)

"Console.Readline()" will print the input out from the user. Whatever the user put inside of the readline lines, it will print it out when the program is started.

EXAMPLE

Syntax:

Console.ReadLine


Output (WriteLine)

The Output (WriteLine) prints a string argument of whatever the user puts into it.

EXAMPLE

Syntax:

Console.WriteLine("")
